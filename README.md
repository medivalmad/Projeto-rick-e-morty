# Projeto-rick-e-morty

# 🌌 Rick and Morty Universe

Um projeto front-end desenvolvido com HTML, CSS e JavaScript puro, consumindo dados em tempo real da API pública de Rick and Morty.

Entre portais verdes, universos caóticos e personagens completamente malucos, esse projeto foi criado com o objetivo de explorar consumo de APIs, manipulação do DOM e criação dinâmica de elementos utilizando JavaScript.

---

# 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Fetch API

---

# 🎯 Objetivo do Projeto

O objetivo deste projeto é desenvolver uma aplicação web capaz de:

* Consumir dados de uma API pública;
* Exibir personagens dinamicamente;
* Trabalhar manipulação do DOM;
* Criar uma interface moderna e responsiva;
* Utilizar JavaScript puro sem frameworks.

---

# 🧪 API Utilizada

API oficial do Rick and Morty:

https://rickandmortyapi.com/

A API fornece:

* Nome dos personagens;
* Imagens;
* Espécie;
* Gênero;
* Status;
* Localização;
* Entre outras informações.
